# Hangry Group Developer Challenge

My name is Natasha Gibe and this is my submission for the Hangry Group challenge!

## Usage
Setup:
```
git clone git@github.com:natbityou/hangry-group-challenge.git
cd hangry-group-challenge
yarn install
```
Run:
```
expo start
```

Expo CLI keyboard shortcuts:
```
Pressing i will open in an iOS simulator.

Pressing a will open in an Android emulator or connected device.

Pressing w will open in your browser. Expo supports all major browsers.
```
